# Cats vs Dogs Classification 🐱🐶

This project is a simple image classification system that distinguishes between cats and dogs using image processing and machine learning techniques.

## Features
- Image preprocessing (resize, normalize)
- Image enhancement (blur, threshold, edges)
- Feature extraction using color histograms
- Machine Learning models:
  - KNN
  - SVM
- Model evaluation and comparison

## Technologies
- Python
- OpenCV
- NumPy
- Matplotlib
- Scikit-learn

## Project Structure
dataset/ → training images  
output_results/ → processed images  
screenshots/ → results and graphs  

## Results
Best accuracy achieved using SVM model.
